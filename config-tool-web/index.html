ⓘ How do Layers, Sticky, Tap & Hold work? ▼
What are Layers?
Think of layers as profiles — each layer is a different set of key assignments. Only one layer is active at a time.
Layer 0 is the default. You can switch to another layer by mapping a key to it.

Example: Power button switches between two profiles
Step 1. Map Power → Layer 1, check Sticky, on Layer 0.
Press Power → switches to Layer 1. Press Power again → Sticky toggles back to Layer 0.
Step 2. Add your Layer 0 mappings (normal remote keys).
Step 3. Add your Layer 1 mappings (e.g. Volume Up → Brightness Up).
Each mapping has layer checkboxes — check which layer(s) it should be active on.

Sticky (Layer targets only)
• Toggle a layer on/off — press once to switch, press again to go back.
Only useful when the output is a Layer. Has no effect on regular key mappings.

Tap & Hold (work with any mapping)
• Tap — fires on quick press-and-release.
• Hold — fires only when held down.
Tip: create two mappings for the same key — one with Tap, one with Hold — to give it dual function.
Example: Tap Power → Mute, Hold Power → switch Layer.
